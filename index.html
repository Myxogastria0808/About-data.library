I develped a R package!


The name is
"data.library".


The R programming language originally provides several data sets. However, there is no package that categorizes the characteristics of these data sets. Therefore, I categorized datasets and created a package that can search datasets by category as a function of package.

This package has two function.


data_library()


It can output summary() values and scatter diagrams of data sets.

help_data_library()


It can display help pages of data sets.

Install
